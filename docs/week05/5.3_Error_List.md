# Known Raw Error Messages to Watch For

The following raw error messages may appear in the dashboard UI or R console if validation and error handling are incomplete. These messages should not be shown directly to non-technical users. They should be replaced with clear user-facing messages.

| ID     | Related Area                | Possible Raw Error / Console Message                                           | Likely Cause                                                                          | Expected User-Friendly Message                                                                                       |
| ------ | --------------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| RAW-01 | HEV / Dependencies          | `Error: there is no package called 'ggnewscale'`                               | The required R package is not installed in the local environment.                     | `The required package ggnewscale is missing. Please install project dependencies before using the HEV plot feature.` |
| RAW-02 | Donor input / fread         | `Error in fread(...): File '...' does not exist or is non-readable`            | Invalid donor-site input has been passed to `fread()` and interpreted as a file path. | `The donor-site input could not be read. Please check the required format and try again.`                            |
| RAW-03 | Donor input / fread         | `Error in fread(input$...)`                                                    | Text box input is passed directly to `fread()` without validation.                    | `The input format is invalid. Please check the donor-site format before importing.`                                  |
| RAW-04 | Donor input / fread         | `Error in fread: Input is empty or only contains whitespace`                   | Empty donor mapping or donor-site input is submitted.                                 | `Please enter donor-site information before importing donor flow data.`                                              |
| RAW-05 | Local file upload / fread   | `Error in fread(file): File is empty`                                          | Uploaded CSV file is empty.                                                           | `The uploaded file is empty. Please upload a file containing valid data.`                                            |
| RAW-06 | Local file upload / fread   | `Error in fread(file): Expected sep but new line, EOF, or other parsing error` | Uploaded CSV has an invalid structure, separator or inconsistent rows.                | `The uploaded file could not be read. Please check that it is a valid CSV file with the expected columns.`           |
| RAW-07 | Local file upload / columns | `undefined columns selected`                                                   | Required columns are missing after reading the uploaded file.                         | `The uploaded file is missing required columns. Please check the file format and upload again.`                      |
| RAW-08 | Metadata / columns          | `object 'biol_site_id' not found`                                              | Metadata file does not contain the required `biol_site_id` column.                    | `The metadata file is missing the required column biol_site_id.`                                                     |
| RAW-09 | Metadata / columns          | `object 'flow_site_id' not found`                                              | Metadata file does not contain the required `flow_site_id` column.                    | `The metadata file is missing the required column flow_site_id.`                                                     |
| RAW-10 | Metadata / columns          | `object 'flow_input' not found`                                                | Metadata file does not contain the required `flow_input` column.                      | `The metadata file is missing the required column flow_input.`                                                       |
| RAW-11 | Data processing             | `object 'flow_data' not found`                                                 | User attempts flow processing before importing flow data.                             | `Please import flow data before running this calculation.`                                                           |
| RAW-12 | Data processing             | `object 'biol_data' not found`                                                 | User attempts invertebrate processing before importing biology data.                  | `Please import biology data before running this calculation.`                                                        |
| RAW-13 | Data processing             | `object 'env_data' not found`                                                  | User attempts RICT prediction before environmental data is available.                 | `Please import environmental data before running RICT predictions.`                                                  |
| RAW-14 | O:E ratio                   | `object 'rict_predictions' not found`                                          | User attempts to calculate O:E ratios before running RICT predictions.                | `Please run RICT predictions before calculating O:E ratios.`                                                         |
| RAW-15 | Join HE data                | `object 'flow_stats' not found`                                                | User attempts to join data before calculating flow statistics.                        | `Please calculate flow statistics before joining biology and flow data.`                                             |
| RAW-16 | Join HE data                | `object 'oe_ratios' not found`                                                 | User attempts to join data before calculating O:E ratios.                             | `Please calculate O:E ratios before joining biology and flow data.`                                                  |
| RAW-17 | HEV                         | `object 'joined_data' not found`                                               | User attempts to create an HEV plot before joining biology and flow data.             | `Please join biology and flow data before creating an HEV plot.`                                                     |
| RAW-18 | Plotting                    | `Error in ggplot(...)`                                                         | Plot input data is missing, empty or invalid.                                         | `The plot could not be created because the required data is missing or invalid.`                                     |
| RAW-19 | Download                    | `cannot open file`                                                             | Dashboard cannot write or access the generated download file.                         | `The file could not be created. Please try again or check file permissions.`                                         |
| RAW-20 | Runtime-generated files     | `cannot open file '27034.csv': No such file or directory`                      | Dashboard attempts to read a generated flow CSV that does not exist.                  | `The required temporary flow file could not be found. Please re-import the flow data.`                               |
| RAW-21 | Runtime-generated files     | `Permission denied`                                                            | Runtime-generated CSV or image file cannot be written or accessed.                    | `The dashboard could not save a temporary file. Please check file permissions or try again.`                         |
| RAW-22 | External data source        | `HTTP error`, `Timeout was reached`, or `cannot open URL`                      | External data source cannot be reached.                                               | `The external data source could not be reached. Please check the connection and try again.`                          |
| RAW-23 | General Shiny               | `Warning: Error in ...`                                                        | Unhandled backend error in a reactive expression.                                     | `Something went wrong while processing this step. Please check your input and try again.`                            |
| RAW-24 | Local path exposure         | `C:/Users/...` or another local file-system path appears in the UI             | A backend error exposes the developer or tester’s local file path.                    | `An internal file-reading error occurred. Please check your input and try again.`                                    |
| RAW-25 | Persistent loading          | No specific message; dashboard remains in loading state                        | Backend processing failed but the UI did not recover properly.                        | `The operation could not be completed. Please check your input and try again.`                                       |

For all raw error cases, the dashboard should:

* Display a clear user-facing message.
* Avoid showing raw R errors, stack traces or local file paths.
* Exit the loading state after the error is handled.
* Keep other pages and controls usable.
* Prevent repeated button clicks while processing is running.
