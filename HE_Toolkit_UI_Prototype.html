Choose your output
Primary path

Create HEV Plot

Visualise the relationship between ecological health and flow variability across monitoring sites.

Requires: Biology · Environmental · Flow data

Calculate O:E Ratio

Compare observed vs. expected ecological scores to assess site health. No flow data required.

Requires: Biology · Environmental data only

Flow Statistics

Compute Q values and seasonal flow summaries from daily gauge records.

Requires: Flow data (NRFA or local upload)

Regression Model

Model ecological response to flow and environmental predictors across sites.

Requires: Joined dataset · O:E · Flow statistics

Prepare / Join Dataset

Import, validate, and join biology, flow, and environmental data into a single analysis-ready dataset.

Requires: Biology · Environmental · Flow · (optional: Water Quality, RHS)
Note Tasks share data behind the scenes — selecting HEV Plot will automatically run O:E ratios and Flow Statistics as intermediate steps. You can review each checkpoint before proceeding.
